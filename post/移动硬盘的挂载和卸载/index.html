移动硬盘的挂载和卸载

2018年5月11日 08:29

有的时候linux不能自动挂载移动硬盘，这时候就需要我们手动挂载移动硬盘。

  1. 插上移动硬盘后运行 sudo fdisk - l 如果有新的硬盘会显示: /dev/sdd1 * 63 1953525167 1953525105 931.5G 7 HPFS/NTFS/exFAT
  2. 创建一个新的文件夹 mkdir temp-disk

  3. 挂载移动硬盘

    sudo mount -t ntfs /dev/sdd1 temp-disk

    进入temp-disk文件夹即可看到移动硬盘中的数据。

  4. 使用完之后卸载移动硬盘

    sudo umount temp-disk
