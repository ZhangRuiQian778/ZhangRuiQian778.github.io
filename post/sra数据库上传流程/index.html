SRA 数据上传主要分为：获取 bioproject accession、biosample accession 和 SRA 测序数据上传。大部分操作在网页上就可以完成，其中两个 Excel 表格的时候需要注意填写和格式问题。

获取 biosample accession

  1. 进入 biosample

  2. 点击 submit 然后新建一个 submission

  3. submitter 填写你的基本信息，标星号的条目为必填项

  4. general information 主要让选择何时公开数据和选择样本数量

  5. sample type 同样根据自己情况选择

  6. attributes 选择网页填写样品信息还是通过 excel 填写后上传，这里我以 excel 填写为例


绿色表示必填选项，蓝色表示至少填一项，黄色表示选填。我们在填写过程中尽量把样品信息描述详细，而且每个样品之间必须不同。如果你把生物重复的样品也单独算作一个 sample 那么建议你在表格最后进行标记，比如：

上传审核之后没问题就可以提交了
7. 提交后等待一会就会收到 biosample accession

获取 bioproject accession

  1. 进入 bioproject 点击 submission 提交一个新的申请

  2. submitter 部分和 biosample 相同

  3. project type 的选择根据自己情况来选择

  4. 选择使用实验材料物种，品种等信息

  5. general info 主要将 public description 部分填一下，其他部分一般需要填写。

  6. 将之前获取的 biosample accession 填上

  7. 我一般在投稿之前就将数据上传，所以这步就跳过了

上面两步完成后就获得了 bioproject accession ，接下来就可以上传数据到 SRA 。

SRA 上传数据

  1. 登陆 SRA ，选择 Log in to submmission portal (for submitting sequence data) 并新建一个 submission

  2. 一开始和 biosample 一样

  3. 填入上面获得 bioproject 号勾选即可公开数据即可

  4. 使用 excel 填写 SRA metadata

    这个表格填写一般没什么问题，按照上面的要求填写即可。如果是双端测序数据那就分别填写 filename 和 filename2 。

  5. 接下来就是将数据上传到服务器。数据量少可以通过网页传输，数据量大建议使用 aspera 或者 FTP 传输。
